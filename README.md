# 파일 위변조 확인

파일 위변조를 확인하기 위한 블록체인 프로젝트 입니다.

1. 지갑기능
2. 파일 업로드(해쉬 추출, 트랜잭션 전송)
3. 파일 리스트 및 조회

의 크게 3가지 기능으로 구분됩니다.




### 소개

1. ##### 사용
```
$ git clone https://github.com/PeterKimG/calcchain.git
$ cd calcchain
$ npm i
$ node index.js
```
